Проекты

УрокOOP1.1_18.02.23

Этот проект реализует симуляцию кассовой системы с поддержкой скидок, управления балансом кошелька и обработки пользовательского ввода.

Данный проект демонстрирует практическое применение основных концепций объектно-ориентированного программирования (ООП) и управления потоками данных, что может быть полезно для кассовых систем и систем управления скидками и кошельками.

Сервис управления пользователями

Проект представляет собой консольное приложение для управления пользователями. Основной класс системы — UserService, который предоставляет функционал для добавления, удаления, поиска пользователей, а также вывода пользователей с балансом выше определенного значения. Проект демонстрирует работу с коллекциями, обработку пользовательского ввода, управление данными о пользователях, а также проверку паролей на соответствие требованиям безопасности.
